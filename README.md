# typescript

## up-and-running-with-typescript

### Install
npm i -g typescript 

### Compile
tsc [file]

#### All exclude config oprions
tsc

#### Complie to different file
tsc --out [tofile] [file]

### Watch
tsc --watch --out [tofile] [file]
tsc -w

### Help
tsc --help

### Config

#### Create
tsc --init
